# StarT 한글 번역 프로젝트

마인크래프트의 GregTech 기반 모드팩인 Star Technology의 한글 번역 모드팩 프로젝트입니다.

# 다운로드 & 적용 방법

## 다운로드 방법

Release에서 자신이 사용 중인 모드팩 버전에 맞는 파일을 다운 받아 해당 파일을 전부 리소스팩 폴더에 넣습니다.

## 적용 방법

리소스팩 설명에 따라, 모드팩을 위한 리소스팩을 위에 우선 적용되게 적용합니다.

# 리소스팩 기여

리소스팩에 기여를 하고 싶으시면 StarT 공식 디코에 있는 한글 번역 스레드에서 기여를 해주시면 됩니다.

# 오역 제보

오역은 StarT 공식 디코에 있는 한글 번역 스레드나 issue 탭에서 받습니다.
